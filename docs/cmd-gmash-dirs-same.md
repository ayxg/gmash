# gmash dirs same

### Usage
`gmash dirs same -p <srcPath> -P <tgtPath>`

### Brief
Get a diff of 2 directories.

### Parameters:
`-p`  `--path  <srcPath>` \
&nbsp;&nbsp;&nbsp;&nbsp;Source path.

`-P`  `--tgt-path  <tgtPath>` \
&nbsp;&nbsp;&nbsp;&nbsp;Target path.
