### Brief
gmash subtree patch -r [repo] -b [branch]

Patch subtree changes to monorepo.

### Parameters:
`-R`  `--remote  [remote = "origin"]` 

`-b`  `--branch  [branch = "main"]` 

`-p`  `--path  <subtreePath>`
