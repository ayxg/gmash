## gmash dirs squash
`gmash dirs squash [path]`

Squash empty directories in the specified path.

### Parameters
- `-p`, `--path [path]` 
    : Target path. Defaults to current.
- `-d`, `--depth [depth]`
    
### Display
- `-h`, `--help` 
    : Display help information about this command.
- `-v`, `--version`
    : [$GMASH_DIRS_SQUASH_VERSION] Display command version.
